# ます形

### 1.動~~ます~~＋たいです    “我想做~”

我想做某事，也能询问对方想不想做某事。

「を」可换成「が」，表达意志更强烈。

「たい」的肯定、否定、时态和「い」形容词一样。

### 2.地点+へ＋動~~ます~~/动作性名词+に行きます    “去某地做某事”

去某地做某事，「に」表目的。后面的动词也可以是「来ます、帰ります」等。

### 3.動~~ます~~＋ましょうか    “我来帮忙吧”

礼貌表达主动为对方做某事。

# て形

### 1.動て＋ください    “请~”

请做某事。不对上。

### 2.動て＋います    “正在~”

1. 表示动作正在进行，用持续动词。
2. 表示动作之后残留下的状态，用瞬间性动词。
3. 表示习惯性行为，长期在做的某事。

### 3.動て＋もいいです    “可以~”

表示可以做某事。

### 4.動て＋はいけません    “不准~”

表示不准做某事。

### 5.动词的授受

1. AはBに動てあげます    “A帮B做某事”

2. AはBに動てもらいます     “A让B帮忙某事”

3. AはBに動てくれます    “A帮B（我方）做某事”

### 6.て表中顿

動て：表示两个以上动作先后发生，时态由句尾的动词决定。

い形くて、な形で、名で：表示并列。

中顿都是顺接。

### 7.動て＋から    “~之后”

前面的动作结束后，进行后面的动作，时态由句尾的动词决定。

### 8.動て＋みます    “试着~一下”

尝试做某个动作。

### 9.動/形/名て＋も    “即使~，也~”

表示逆接，即使有前项的情况，后项也仍旧如此。

强调时可在句首加「いくら、たとえ、もし」等。

注意和「~でも」区分，表示「~でも」“甚至/连~”。

# ない形

### 1.動~~ない~~＋ないでください    “请不要~”

请求或指示对方不要做什么事。

### 2.動~~ない~~＋なければなりません/ないといけません    “必须/不得不”

前半断表示假设，后半段表示否定。

### 3.動~~ない~~＋なくてもいいです    “不~也可以”

表示没有必要进行某个行为。

### 4.動~~ない~~/形/名＋なくても    “即使不~”

い形い变くない、な形/名＋じゃない。

表示否定的逆接。

### 5.動ない＋と

表示否定条件，“如果不~，就~”。

# 辞書形

### 1.わたしの趣味/夢は＋「動る＋こと/名詞」＋です    “我的爱好/梦想是~”

動る＋こと可以将动词变成名词。

### 2.「動る＋こと/名詞」＋ができます    “能/会~”

表示能力或可能。

### 3.「動る/名詞＋の/期间数量词」＋前に    “~之前”

表示做某事之前如何。

### 4.動る＋と

1. “一~，就（必然）会~”    表必然发生，只要发生前项，就一定会有后项。
2. “~之后”    表相继进行，表示相继进行前后两个动作，动作是同一主体，后项多用过去式。

# た形

### 1.動た＋ことがあります    “曾经~过”

表示曾经有过某种经验或体验。

如果只是单纯想表达做了某事，用过去时即可。

### 2.「動た/名＋の」＋後（で）    “~之后”

强调做完了前面的事之后再做后面的事。

比「てから」更强调时间先后。

### 3.～動た＋り、～動た＋りします    “又~又~”

表示动作的列举，时态在句尾「します」体现。

句尾「します」可接续其他句型。

### 4.動た＋ら    “~之后”

强调前面的动作完成之后经行其他动作，是一种既定的条件表达，前项已发生或确定要发生，后项一般不用过去式。

### 5.たら    “如果~的话”

接续：動/形/名た＋ら

表示假定条件，假设前项发生，即会发生后项。句子前接「もし」更强调假设。

# 简体

### 1.简体用于对话

1. 可省略助词「は、が、を、へ」
2. 疑问句省略句尾「か、だ」
3. 「けど」代替「が」表转折
4. 「たって」「だって」代替「ても」「でも」
5. 「て」「ないで」代替「てください」「ないでください」
6. 「てる」代替「ている」

### 2.と言います    “某人说”

表示引用，直接引用需加引号，间接引用前接简体。

引用句时态不变。

### 3.と思います    “我想/我认为/我觉得”

表示推量、意见、想法。

前接简体。

どう思いますか？你认为怎么样呢？

### 4.でしょう？    “~吧？”

表示希望对方同意自己。

「でしょう」的简体是「だとう」。

前接简体，有だ去だ。

# 其他

### 1.なります＆します    “变成/弄成”

前接：い形い变く、な形/名＋に

なります表自然变化；します表人为变化。

### 2.名词性修饰+とき    “~的时候”

名词性修饰：动词/い形简体、な形＋な、名＋の

表达在做前项事情的时候做后项。

とき前为动词原型时表示动作尚未介绍，た形表示已经介绍。

とき前动词时态不受句尾影响。
